# Python Slideshow Project

This is a simple image slideshow application built using Python and Tkinter.
The slideshow displays images automatically after a fixed interval.

## Features
- Image slideshow using Tkinter
- Start slideshow button
- Image changes every 3 seconds
- Simple and beginner-friendly project

## Technologies Used
- Python
- Tkinter (GUI library)

## How to Run the Project
1. Make sure Python is installed on your system
2. Open terminal in project folder
3. Run the following command:

   python slideshow.py

## Project Structure
- slideshow.py   -> Main Python file
- images/        -> Folder containing slideshow images
- README.md      -> Project description file

## Output
A window opens showing images as a slideshow.
